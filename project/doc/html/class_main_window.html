Detailed Description

The MainWindow class, inherits from QMainWindow and uses the macro Q_OBJECTS in order to call signals and slots Note: The custom messages need to be modified such that they are generated from the packages turtlebot_ui, since there is no individual detect_marker package anymore. Every additional package that is to be included in the user interfaces would get a folder inside turtlebot_ui/src.
